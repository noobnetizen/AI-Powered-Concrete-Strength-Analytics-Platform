# 🏗️ AI-Powered Concrete Strength Analytics Platform


An AI-powered concrete strength prediction platform that estimates **7-Day** and **28-Day Compressive Strength** using **XGBoost Machine Learning Models** and **SHAP Explainable AI**. The application provides an intuitive interface for engineers, researchers, and students to analyze concrete mix compositions and generate accurate strength predictions.

# 📖 About the Project

Concrete compressive strength is one of the most important parameters in construction and civil engineering. Traditional strength testing requires curing periods and destructive laboratory testing, making the process time-consuming and resource-intensive.

This project leverages Machine Learning and Explainable AI to predict concrete compressive strength using material composition and mix design parameters.

The machine learning workflow includes:

* Exploratory Data Analysis (EDA)
* Feature Engineering
* XGBoost Model Training
* Model Evaluation
* SHAP Explainability Analysis
* Prediction of 7-Day and 28-Day Compressive Strength

The user interface was developed using **Jotform AI App Builder**, while the complete machine learning pipeline was built using Python and XGBoost.

# 🧰 Installation & Setup

## Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/AI-Powered-Concrete-Strength-Analytics-Platform.git

cd AI-Powered-Concrete-Strength-Analytics-Platform
```

## Create Virtual Environment

```bash
python -m venv venv

venv\Scripts\activate
```

## Install Dependencies

```bash
pip install -r requirements.txt
```

## Run Application

```bash
python app.py
```

Application will run at:

```text
http://127.0.0.1:5000
```
